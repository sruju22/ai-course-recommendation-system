# 🚀 AI Course Recommendation System

An intelligent online learning platform that recommends courses using Machine Learning (KNN algorithm) based on user learning behavior and progress.

---

## 🧠 Features

- Personalized course recommendations
- KNN-based machine learning model
- Category-aware recommendations
- Fallback system when ML fails
- User progress tracking
- Clean UI with React

---

## 🛠️ Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Machine Learning:
- Python
- Flask API
- Scikit-learn (KNN)

Database:
- MySQL

---

## ⚙️ How It Works

1. User learning data is collected (courses completed, quiz scores)
2. KNN algorithm finds similar users
3. Recommends courses based on similarity
4. Filters by category for better relevance
5. Uses fallback if no ML result
